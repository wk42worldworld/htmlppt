# 사람을 위한 튜토리얼

이 문서는 코드를 쓰지 않고 `.ppt.html` 프레젠테이션을 만들고 싶은 사용자를 위한 안내입니다. 완성된 파일은 재생, 공유, 재편집이 가능합니다.

## 1. 편집기 열기

로컬에서 실행합니다.

```bash
npm install
npm run serve
```

다음을 엽니다.

```text
http://localhost:5173
```

GitHub Releases에서 데스크톱 앱을 내려받아 바로 실행할 수도 있습니다.

## 2. 첫 슬라이드 수정

왼쪽에서 1번 슬라이드를 선택하고 오른쪽 패널에서 수정합니다.

- 문서 이름
- 테마
- 키커
- 제목
- 부제목

가운데 슬라이드 캔버스가 즉시 업데이트됩니다.

## 3. 슬라이드 추가와 정렬

왼쪽 슬라이드 목록에서 다음을 할 수 있습니다.

- `+` 로 슬라이드 추가
- 썸네일 클릭으로 슬라이드 전환
- 복제, 위로, 아래로, 삭제로 순서 관리

각 슬라이드는 표지, 비교, 세 카드, 데이터, 표, 코드 등의 레이아웃을 사용할 수 있습니다.

## 4. AI JSON 가져오기

상단의 `AI JSON` 을 클릭하고 AI가 생성한 deck JSON을 붙여넣은 뒤 `从文本载入` 을 클릭합니다.

최소 예시:

```json
{
  "version": "0.1",
  "title": "내 프레젠테이션",
  "theme": "launch",
  "aspectRatio": "16:9",
  "slides": [
    {
      "id": "slide-1",
      "layout": "hero",
      "title": "첫 슬라이드",
      "subtitle": "이것은 PPT.html 입니다"
    }
  ]
}
```

## 5. 발표하기

상단의 `演示` 를 클릭해 발표 모드로 들어갑니다.

발표 중에는 다음을 사용할 수 있습니다.

- 이전, 다음 버튼
- 키보드 좌우 방향키
- 종료 버튼으로 편집기로 돌아가기

## 6. 단일 파일 내보내기

`下载 PPT.html` 을 클릭합니다. 내려받은 파일은 일반 HTML 파일입니다.

- 브라우저에서 열어 재생
- 다른 사람에게 공유
- PPT.html Studio에서 다시 열어 편집
