# 普通用户教程

这个教程面向不写代码的用户，目标是完成一份可以播放、可以分享、可以再次编辑的 `.ppt.html` 演示文稿。

## 1. 打开编辑器

本地运行：

```bash
npm install
npm run serve
```

然后打开：

```text
http://localhost:5173
```

也可以下载 Release 里的桌面程序，直接启动 PPT.html Studio。

## 2. 从模板开始

点击顶部 `模板`，选择产品发布、课程课件、项目汇报或 demo deck。模板会生成一份完整草稿，你只需要替换标题、内容和数据。

## 3. 修改第一页

在左侧选择第 1 页，在右侧属性面板修改：

- 文稿名称
- 主题
- 眉标
- 标题
- 副标题

中间画布会实时更新。

## 4. 添加和调整页面

左侧页面区可以：

- 点击 `+` 添加页面
- 点击页面缩略图切换页面
- 使用复制、上移、下移、删除管理页面顺序

每一页都可以在右侧选择版式，例如封面、对比、三卡片、数据、表格、代码。

图片页可以点击 `选择本地图片`，图片会嵌入到 `.ppt.html` 文件中。

## 5. 用 AI JSON 导入内容

点击顶部 `AI JSON`，把 AI 生成的 deck JSON 粘贴进去，然后点击 `从文本载入`。

最小示例：

```json
{
  "version": "0.1",
  "title": "我的演示",
  "theme": "launch",
  "aspectRatio": "16:9",
  "slides": [
    {
      "id": "slide-1",
      "layout": "hero",
      "title": "第一页标题",
      "subtitle": "这是一份 PPT.html"
    }
  ]
}
```

导入后点击 `检查`。如果报告里有错误或警告，可以复制报告给 AI，让它按报告修复 JSON。

## 6. 演示播放

点击顶部 `演示` 进入播放模式。

播放时可以：

- 点击上一页、下一页
- 使用键盘左右方向键
- 点击退出回到编辑器

## 7. 保存和分享

浏览器版点击 `保存 / 下载`。桌面版可以使用 `保存`、`另存为` 或快捷键 `Ctrl/Cmd+S`。得到的文件是一个普通 HTML 文件，可以：

- 直接用浏览器打开播放
- 发给别人
- 再次用 PPT.html Studio 打开编辑
