# PPT.html 大模块自由插入计划

本文档把“常见 PPT 模块必须能插入到当前页，并像 Keynote / PowerPoint 元素一样编辑”的需求拆成可执行阶段。目标不是让用户学习 HTML 或 JSON，而是让 AI 写结构化对象，人类在画布上点击、拖拽、双击和右键完成编辑。

## 产品目标

- 点击或拖拽一个模块时，默认插入到当前页的画布中，不新建页面，不覆盖现有内容。
- 每个模块都是 `slide.objects[]` 里的结构化对象，保存位置、尺寸、层级、样式和内容数据。
- 人类可以直接选中、拖动、缩放、复制、粘贴、删除、对齐、分布、置顶置底。
- 人类不需要编辑 JSON 就能完成常见内容修改；JSON 面板只作为高级入口。
- 导出的 `.ppt.html` 是单文件，别人只要用软件或浏览器打开，就能看到完整内容。
- AI 能按稳定 schema 生成和局部修改这些对象，不写自由 HTML。

## P0 发布必需模块

| 模块 | 当前基础 | 必须补齐 | 验收标准 |
| --- | --- | --- | --- |
| 文本框 | 已支持插入、双击编辑、拖拽缩放 | 继续保证插入后立刻可编辑，删除/复制/样式稳定 | 新文本框可见、可删、可保存重开 |
| 图片 | 已支持图片对象和单文件打包 | 替换、裁切/适应、caption、alt、拖拽不卡顿 | 插入本地图片后导出仍完整 |
| 视频 | 已支持视频对象和单文件打包 | poster、controls、loop、autoplay、fit | 播放模式可播放，编辑模式不误显示提示文案 |
| 音频 | 已支持音频对象和单文件打包 | caption、controls、替换资源 | 保存重开后音频仍可播放 |
| 图表 | 已有柱状/折线/环形渲染 | 专用图表数据编辑器，支持增删系列和数据点 | 不碰 JSON 完成 2 系列图表 |
| 表格 | 已有对象表格和页面表格 | 右键增删行列、清空单元格、选中单元格不误删整表 | 双击单元格编辑，右键能插入行列 |
| 形状/线条/箭头/标注 | 已有 shape 对象 | 填充、描边、线宽、文字、箭头方向 | 线条缩放方向稳定，标注文字可编辑 |
| 卡片组 | 已有三卡渲染 | 2/3/4 卡内容编辑和增删卡片 | 长文本不裁切，不需要 JSON |
| 指标/KPI | 已有 3 指标渲染 | 数值、标签、说明、趋势/状态色编辑 | 大数字不撑破布局 |
| 时间线/流程 | 已有 items 渲染 | 3-5 步增删和横向/流程变体 | 双击步骤标题/正文可改 |
| 对比/决策 | 已有 left/right 渲染 | 胜出标签、结论、左右内容原位编辑 | 插入后留在当前页 |
| 引用/代码 | 已有对象渲染 | 作者、语言、代码内容专用编辑 | 代码换行和等宽样式保留 |

## P1 常见商业 PPT 模块

- 目录 / Agenda：章节号、标题、状态、当前章节高亮。
- 图片网格 / Gallery：2、3、4、6 图，每格支持本地图片、caption、fit。
- 图文列表 / Icon List：3-6 项，图标、标题、正文。
- 2x2 矩阵 / SWOT：象限标题、正文、重点标记。
- 漏斗 / 金字塔：3-6 层，适合转化、优先级、战略层级。
- Logo strip / 客户墙：多 logo 图片格，统一适配。
- 组织结构 / 团队：头像、姓名、角色、汇报关系。
- 设备截图框：浏览器框、手机框、图片填充。
- 下一步 / Action items：事项、负责人、日期、状态。

## 工程步骤

1. 建立对象模块注册表：统一声明 `type`、`variant`、默认尺寸、默认数据、Inspector 字段、可编辑子路径和测试 fixture。
2. 稳定 P0 对象基础体验：当前页插入、选中、拖拽、缩放、复制、粘贴、删除、右键、层级、对齐。
3. 先做表格体验闭环：表格单元格和整表选择分开，右键支持上下左右插入、删除行列、清空单元格，导出不截断。
4. 增加 Typed Object Inspector：媒体、图表、表格、形状、卡片、指标、时间线、对比模块都有专用编辑区。
5. 增加对象内部原位编辑路径：例如 `objects.2.data.cards.0.title`、`objects.3.data.items.1.text`。
6. 双渲染验收：编辑器 renderer 和 standalone `.ppt.html` 导出都要支持同一模块。
7. 更新 schema、spec、AI 教程、人类教程和四国语言首页。

## 验收清单

- 点击任意 P0 模块，当前页页数不变，`objects.length` 增加 1。
- 拖拽模块到画布，落点接近鼠标释放位置。
- 选中对象后可 Delete 删除，选中表格单元格时 Delete 清空单元格或只删除选中内容，不误删整表。
- 右键对象可复制、粘贴、重复、调整层级、删除。
- 右键表格单元格可在上/下/左/右插入、删除行列、清空单元格。
- 双击可编辑文本类内容，提交后保存到结构化路径。
- 保存 `.ppt.html` 后重新打开，模块数量、位置、尺寸、层级、样式、数据一致。
- 播放模式不显示任何编辑提示文字。
- 竖屏手机播放没有黑边以外的异常遮挡，底部控制条默认隐藏，移动鼠标或触摸才显示。
