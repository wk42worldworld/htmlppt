# 人間向けチュートリアル

このガイドは、コードを書かずに `.ppt.html` プレゼンテーションを作りたいユーザー向けです。完成したファイルは再生、共有、再編集できます。

## 1. エディタを開く

ローカルで実行します。

```bash
npm install
npm run serve
```

次を開きます。

```text
http://localhost:5173
```

GitHub Releases からデスクトップアプリをダウンロードして起動することもできます。

## 2. テンプレートから始める

上部の `模板` をクリックし、製品ピッチ、授業資料、プロジェクト報告、demo deck を選びます。テンプレートから完全な下書きが作られるので、内容を置き換えて使えます。

## 3. 1 枚目を編集する

左側で 1 枚目を選び、右側のパネルで編集します。

- デッキ名
- テーマ
- キッカー
- タイトル
- サブタイトル

中央のスライドはすぐに更新されます。

## 4. スライドを追加・並べ替える

左側のスライド一覧で操作できます。

- `+` でスライドを追加
- サムネイルをクリックして切り替え
- 複製、上へ、下へ、削除で順序を管理

各スライドでは、表紙、比較、3 カード、データ、表、コードなどのレイアウトを選べます。

画像スライドでは `选择本地图片` をクリックすると、画像を `.ppt.html` ファイル内に埋め込めます。

## 5. AI JSON を読み込む

上部の `AI JSON` をクリックし、AI が生成した deck JSON を貼り付け、`From text` で読み込みます。

最小例：

```json
{
  "version": "0.1",
  "title": "私のデッキ",
  "theme": "launch",
  "aspectRatio": "16:9",
  "slides": [
    {
      "id": "slide-1",
      "layout": "hero",
      "title": "最初のスライド",
      "subtitle": "これは PPT.html です"
    }
  ]
}
```

読み込み後に `检查` をクリックします。エラーや警告があれば、レポートを AI に渡して JSON を修正できます。

## 6. プレゼンする

上部の `演示` をクリックして再生モードに入ります。

再生中は次の操作ができます。

- 前へ、次へをクリック
- 左右矢印キーを使用
- 退出でエディタに戻る

## 7. 保存して共有する

ブラウザ版では `保存 / 下载` をクリックします。デスクトップ版では `保存`、`另存为`、または `Ctrl/Cmd+S` を使います。保存されたファイルは通常の HTML です。

- ブラウザで開いて再生
- 他の人に送信
- PPT.html Studio で再度開いて編集
