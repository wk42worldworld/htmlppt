# 人間向けチュートリアル

このガイドは、コードを書かずに `.ppt.html` プレゼンテーションを作りたいユーザー向けです。完成したファイルは再生、共有、再編集できます。

## 1. エディタを開く

ローカルで実行します。

```bash
npm install
npm run serve
```

次を開きます。

```text
http://localhost:5173
```

GitHub Releases からデスクトップアプリをダウンロードして起動することもできます。

## 2. 1 枚目を編集する

左側で 1 枚目を選び、右側のパネルで編集します。

- デッキ名
- テーマ
- キッカー
- タイトル
- サブタイトル

中央のスライドはすぐに更新されます。

## 3. スライドを追加・並べ替える

左側のスライド一覧で操作できます。

- `+` でスライドを追加
- サムネイルをクリックして切り替え
- 複製、上へ、下へ、削除で順序を管理

各スライドでは、表紙、比較、3 カード、データ、表、コードなどのレイアウトを選べます。

## 4. AI JSON を読み込む

上部の `AI JSON` をクリックし、AI が生成した deck JSON を貼り付け、`From text` で読み込みます。

最小例：

```json
{
  "version": "0.1",
  "title": "私のデッキ",
  "theme": "launch",
  "aspectRatio": "16:9",
  "slides": [
    {
      "id": "slide-1",
      "layout": "hero",
      "title": "最初のスライド",
      "subtitle": "これは PPT.html です"
    }
  ]
}
```

## 5. プレゼンする

上部の `演示` をクリックして再生モードに入ります。

再生中は次の操作ができます。

- 前へ、次へをクリック
- 左右矢印キーを使用
- 退出でエディタに戻る

## 6. 単一ファイルを書き出す

`下载 PPT.html` をクリックします。ダウンロードしたファイルは通常の HTML です。

- ブラウザで開いて再生
- 他の人に送信
- PPT.html Studio で再度開いて編集
