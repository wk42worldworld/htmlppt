# PPT.html Studio

PPT.html Studio 是一个面向 AI 时代的演示文稿工具。它不让 AI 直接写复杂 PPTX，也不要求人类写 HTML，而是使用一种结构化的 `.ppt.html` 文件：AI 生成 JSON，软件渲染成漂亮幻灯片，人类继续可视化编辑。

## 新版介绍

v0.2.5 参考 OpenHuman 这类本地优先桌面 AI 应用的干净界面语言，重做了编辑器整体视觉：深色紧凑顶栏、更轻的左右面板、更精致的缩略图、表单、按钮和弹窗、更安静的画布网格，以及更好的窄屏响应式表现。

v0.2.4 在第一版画布编辑基础上增加了更完整的选择与变形控制：点击可编辑元素会显示选中框，可以拖拽移动、用八个手柄缩放、用方向键微调、按 `Shift` 加速微调，并可重置当前元素的位置和尺寸。所有结果都会保存为结构化 `canvas` 的 `x/y/w/h` 数据，导出的 `.ppt.html` 单文件也能保持。

v0.2.3 增加了第一版画布式直观编辑：可以在画布上双击标题、正文、卡片、数据、表格单元格、图表图例和代码块直接修改；可以拖动画布元素做轻量位置微调，位置会保存为结构化 `canvas` 偏移；也可以把图片拖入画布、拖动左侧缩略图调整页面顺序。

v0.2.2 改进了全屏演示缩放，16:9 屏幕下幻灯片会真正铺满，不再出现旧的黑边。v0.2.1 增加了结构化图表页：柱状图、折线图、环形图既可以在编辑器里可视化编辑，也可以由 AI 生成 `chart` JSON。

它的核心定位是：AI 写结构，人类做判断，渲染器负责稳定排版。

## 适合谁

- 用 AI 生成汇报、课程、路演和产品介绍的人
- 希望文件能离线播放、直接分享、再次编辑的人
- 不想手工调整一堆文本框和坐标的团队

## 核心能力

- 单文件 `.ppt.html` 分享和播放
- 桌面端打开、保存、另存为 `.ppt.html`
- 产品发布、课程课件、项目汇报等起稿模板
- 可视化编辑标题、正文、图片、对比、卡片、数据、图表、表格、代码
- 画布上双击文字直接修改，拖动和缩放可编辑元素，用方向键微调，拖动缩略图调整页面顺序
- 图片支持本地嵌入、裁切/完整显示、说明文字、全屏图和背景图文版式
- 图表支持用结构化标签和数据系列生成柱状图、折线图、环形图
- AI JSON 导入
- 质量检查报告，可直接复制给 AI 修复
- 16 种基础版式和 4 套主题
- GitHub Actions 自动构建 Linux、macOS arm64、macOS x64、Windows 程序，并支持 macOS 签名与公证配置

## 快速开始

```bash
npm install
npm run serve
```

打开 `http://localhost:5173`。

## 五分钟教程

1. 打开 PPT.html Studio。
2. 点击 `模板`，选择产品发布、课程课件、项目汇报或 demo deck。
3. 在右侧面板修改标题、副标题、数据、图表、表格和备注，也可以直接在画布上双击文字修改、拖动元素、缩放元素和键盘微调。
4. 需要 AI 起稿时，点击 `AI JSON`，直接粘贴 AI 输出的 JSON 或带 `json` 围栏的代码块。
5. 点击 `检查`，把检查报告复制给 AI 修复错误或警告。
6. 点击 `演示` 预览播放效果。
7. 浏览器版点击 `保存 / 下载`；桌面版使用 `保存` 或 `另存为`，得到一个可分享的 `.ppt.html` 文件。
