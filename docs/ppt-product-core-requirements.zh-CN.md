# PPT.html 产品化核心需求与实现任务

本文档聚焦把 PPT.html Studio 做成普通用户可以像 Keynote 或 PowerPoint 一样上手的演示文稿产品。原则是：用户在当前页面上插入、选择、拖拽、缩放和编辑对象；模板只用于新建页面或快速版式，不替代插入。

## 产品原则

- 当前页优先：点击“图表、图片、表格、形状、视频”等入口时，默认把对象插入当前页，不自动新建页面。
- 操作接近 PPT/Keynote：对象出现在画布上，用户可以直接拖动、缩放、复制、删除、改样式。
- 低学习成本：常见动作使用熟悉图标、右侧属性面板、上下文菜单和快捷键，不要求用户理解 JSON 或 HTML。
- AI 友好但不暴露复杂性：文件内部保存结构化对象数据，AI 可以稳定修改；普通用户看到的是可视化对象。
- 兼容旧文件：旧版 `layout`、`chart`、`table`、`image` 等页面级数据仍可打开；新增对象层不破坏既有文稿。

## MVP 必须具备

### 1. 当前页对象插入

- [x] 点击“图片”时，在当前页插入图片对象并打开文件选择，不新建页面。
- [x] 点击“图表”时，在当前页插入图表对象，不新建页面。
- [x] 点击“表格”时，在当前页插入表格对象，不新建页面。
- [x] 点击“视频/音频”时，在当前页插入媒体对象，不新建页面。
- [x] 点击“文本”时继续在当前页插入文本框。
- [x] 拖拽组件到画布时，对象落在拖拽位置。

### 2. 统一对象模型

- [x] 每页支持 `objects` 数组。
- [x] 每个对象包含 `id`、`type`、`x`、`y`、`w`、`h`、`rotation`、`zIndex`、`data`、`style`。
- [x] 对象可以保存为 `.ppt.html` 单文件，AI 可以继续读取和编辑。
- [x] 旧页面级字段继续渲染，新对象层叠加在其上。

建议数据结构：

```json
{
  "objects": [
    {
      "id": "obj_1",
      "type": "chart",
      "x": 120,
      "y": 180,
      "w": 520,
      "h": 300,
      "rotation": 0,
      "zIndex": 3,
      "data": {
        "kind": "bar",
        "labels": ["Q1", "Q2", "Q3"],
        "series": [{ "name": "收入", "values": [12, 20, 31] }]
      },
      "style": {}
    }
  ]
}
```

### 3. 对象选择与编辑

- [x] 单击对象显示选择框。
- [x] 拖动对象改变位置。
- [x] 八个控制点缩放对象。
- [x] Delete/Backspace 删除对象。
- [x] Cmd/Ctrl+C、Cmd/Ctrl+V 复制粘贴对象。
- [x] 方向键微调位置，Shift 加速。
- [x] Shift 点击支持多选；多选后可批量方向键移动和 Delete 删除。
- [x] 右侧属性面板显示对象位置、尺寸、旋转、层级和通用对象数据。

### 4. 图表对象

- [x] 支持柱状图、折线图、环形图作为当前页对象。
- [ ] 部分支持：右侧对象面板可通过对象数据 JSON 编辑图表类型、标签、数据系列和单位；后续补更直观的专用表单。
- [x] 图表对象可以拖拽、缩放、删除。
- [x] 图表对象可以复制粘贴。
- [x] 图表默认尺寸适合当前页，不遮挡标题。

### 5. 图片和媒体对象

- [x] 图片对象支持本地文件插入，并打包进单文件。
- [x] 图片支持填充裁切、完整显示、圆角、边框、透明度。
- [x] 视频对象支持本地文件插入、封面、填充/完整显示。
- [x] 音频对象支持本地文件插入和基础播放控件。

### 6. 表格对象

- [x] 表格作为当前页对象插入。
- [ ] 部分支持：支持通过对象数据 JSON 编辑列、行和单元格文本；后续补表格式编辑体验。
- [x] 支持基础样式：字号、颜色、边框、表头强调。

## 第二阶段

- [x] 形状：矩形、圆形、线条、箭头、圆角矩形、标注框。
- [x] 对齐与分布：左对齐、居中、右对齐、顶对齐、水平/垂直分布。
- [x] 层级：置顶、置底、上移一层、下移一层。
- [ ] 部分支持：已支持 Shift 多选、批量移动、批量删除；框选、组合、取消组合待做。
- [ ] 参考线与吸附：中心线、边缘吸附、等距吸附。
- [ ] 部分支持：对象右键菜单已支持复制、粘贴、重复、删除、层级；锁定、对齐待做。

## 第三阶段

- [ ] 对象动画：进入、强调、退出、按点击顺序播放。
- [ ] 演讲者视图：备注、下一页预览、计时。
- [ ] 母版：统一字体、页码、页脚、品牌元素、默认占位符。
- [ ] 导出增强：PDF、图片、PPTX/ODP 兼容路线。
- [ ] AI 编辑对象：让 AI 按对象 ID 修改当前页，不重写整页。

- [x] 在第 1 页点击“图片”，页面数量不变，当前页出现图片对象并可选择本地图片。
- [x] 拖拽“表格”到画布中间，表格对象出现在落点附近。
- [x] 新对象可以拖动、缩放、删除。
- [x] 新对象可以复制、粘贴、重复和调整层级。
- [x] Shift 多选两个对象后可以批量方向键移动、顶/底/左/右/居中对齐。
- [x] 保存为 `.ppt.html` 后 standalone 播放器可以渲染对象。
- [x] 旧版页面级 chart/table/image 文稿仍能正常打开和播放。
