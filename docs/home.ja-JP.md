# PPT.html Studio

PPT.html Studio は AI 時代のためのプレゼンテーションツールです。AI に壊れやすい PPTX を直接書かせるのではなく、人間に HTML を書かせるのでもありません。`.ppt.html` ファイルの中に構造化 JSON を保存し、ブラウザで再生できるスライドとして表示し、アプリで再編集できます。

## 新機能

v0.2.9 では、最初の要素スタイルパネルを追加しました。キャンバス上のタイトル、本文、テキストボックス、カード、表セル、指標、グラフ凡例、メディアブロックをクリックすると、右側パネルで文字サイズ、文字色、配置、太字、斜体、背景、枠線、角丸、不透明度を調整できます。結果は構造化された `styles[path]` データとして保存され、書き出した `.ppt.html` にも残り、AI も読み取って編集を続けられます。

v0.2.8 では、テキスト挿入と編集の滑らかさを改善しました。`テキスト` をクリックすると、テキストレイアウトへ切り替わるのではなく、現在のスライドに見えるキャンバステキストボックスを追加します。テキストボックスは既定でタイトルを避けた位置に置かれ、すぐに編集、ドラッグ、リサイズ、矢印キーでの微調整、Delete/Backspace での削除ができ、構造化された `textBoxes` データとして `.ppt.html` に保存されます。右側パネルの入力もフレーム単位でまとめて描画し、下書き保存を遅延させることで、より滑らかになりました。

v0.2.7 では `.ppt.html` を本当の単一ファイルコンテナとして扱います。保存またはダウンロードの前に、外部画像、動画、動画ポスター、音声を Data URI として同じ HTML に埋め込みます。音声も、挿入、ドラッグ、ダブルクリック差し替えができる正式なコンポーネントになりました。

v0.2.6 では、メインエディタに見えるコンポーネントパレットを追加しました。テキスト、画像、動画、グラフ、表、カード、指標、タイムライン、引用、コードをクリックまたはドラッグで現在のスライドに挿入できます。主要なツールバー操作はアイコン中心になり、hover で説明が表示されます。動画は正式なスライド内容として扱われ、アプリ UI は中国語、英語、日本語、韓国語に切り替えられます。

v0.2.5 では、OpenHuman のようなローカルファーストのデスクトップ AI アプリの清潔な UI を参考に、エディタ全体の見た目を更新しました。コンパクトな黒いトップバー、軽い左右パネル、洗練したサムネイル、フォーム、ボタン、ダイアログ、落ち着いたキャンバスグリッド、柔らかい影、狭い画面での表示改善を含みます。

v0.2.4 ではキャンバス編集に選択と変形の操作を追加しました。編集可能な要素をクリックすると選択枠が表示され、ドラッグ移動、8 つのハンドルによるサイズ変更、矢印キーでの微調整、`Shift` による高速微調整、位置とサイズのリセットができます。結果は構造化された `canvas` の `x/y/w/h` データとして保存され、書き出した `.ppt.html` でも維持されます。

v0.2.3 では最初のキャンバス型編集レイヤーを追加しました。キャンバス上でタイトル、本文、カード、指標、表のセル、グラフ凡例、コードブロックをダブルクリックして直接編集できます。編集可能な要素は軽くドラッグして位置を微調整でき、その位置は構造化された `canvas` オフセットとして保存されます。画像をキャンバスへドロップしたり、サムネイルをドラッグしてスライド順を変えたりすることもできます。

v0.2.2 では全画面プレゼンテーションの縮放を改善し、16:9 画面では古い黒い余白なしでスライドが画面いっぱいに表示されます。v0.2.1 では構造化グラフスライドを追加しました。棒、折れ線、ドーナツグラフを視覚的に編集でき、AI も同じ `chart` JSON を生成できます。

基本方針は、AI が構造を書き、人間が判断し、レンダラーが安定したレイアウトを保つことです。

## 対象ユーザー

- AI でレポート、授業資料、ピッチ資料、製品紹介を作る人
- オフラインで開けて、共有できて、再編集できるファイルが必要なチーム
- 多数のテキストボックスを手作業で配置したくないユーザー

## 主な機能

- 単一ファイル `.ppt.html` の再生と共有。画像、動画、音声、ポスター、CSS、プレイヤー JavaScript を 1 ファイルに同梱
- デスクトップアプリで `.ppt.html` を開く、保存、別名保存
- 製品ピッチ、授業資料、プロジェクト報告の開始テンプレート
- タイトル、本文、画像、比較、カード、データ、グラフ、表、コードの視覚編集
- メイン画面のコンポーネントパレットから、テキスト、画像、動画、音声に加え、比較、グラフ、表、カード、指標、タイムラインの複数バリエーションをクリックまたはドラッグで挿入
- キャンバス上でのダブルクリック編集、要素のドラッグ移動とサイズ変更、キーボード微調整、画像ドロップ、サムネイルのドラッグ並べ替え
- 画像はローカル埋め込み、cover/contain、キャプション、全画面画像、背景画像レイアウトに対応
- 動画はローカル埋め込み、URL / Data URI、ポスター、キャプション、cover/contain に対応
- 音声はローカル埋め込み、URL / Data URI、キャプション、ブラウザ標準コントロールに対応
- グラフは構造化されたラベルと系列から棒、折れ線、ドーナツを描画
- プレゼンモードと書き出した単一ファイルプレイヤーは、なし、フェード、スライド、プッシュ、ズームの切替効果に対応
- アプリ UI は中国語、英語、日本語、韓国語に切り替え可能
- AI JSON インポート
- 人間にも AI にも読める検証レポート
- 17 種類の基本レイアウトと 4 種類のテーマ
- GitHub Actions による Linux、macOS arm64、macOS x64、Windows ビルド。macOS 署名と公証も設定可能

## クイックスタート

```bash
npm install
npm run serve
```

`http://localhost:5173` を開きます。

## 5 分チュートリアル

1. PPT.html Studio を開きます。
2. `模板` をクリックし、製品ピッチ、授業資料、プロジェクト報告、または demo deck を選びます。
3. 右側のパネルでタイトル、サブタイトル、指標、グラフ、表、画像、メモを編集します。キャンバス上では直接ダブルクリック、ドラッグ、サイズ変更、キー微調整もできます。
4. AI の下書きを使う場合は `AI JSON` をクリックします。通常の JSON と `json` 囲みコードブロックの両方を読み込めます。
5. `检查` をクリックします。エラーや警告があれば、検証レポートを AI に渡して修正できます。
6. `演示` でプレビューします。
7. ブラウザ版では `保存 / 下载`、デスクトップ版では `保存` / `另存为` を使い、共有可能な `.ppt.html` ファイルを作成します。
