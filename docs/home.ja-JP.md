# PPT.html Studio

PPT.html Studio は AI 時代のためのプレゼンテーションツールです。AI に壊れやすい PPTX を直接書かせるのではなく、人間に HTML を書かせるのでもありません。`.ppt.html` ファイルの中に構造化 JSON を保存し、ブラウザで再生できるスライドとして表示し、アプリで再編集できます。

## 対象ユーザー

- AI でレポート、授業資料、ピッチ資料、製品紹介を作る人
- オフラインで開けて、共有できて、再編集できるファイルが必要なチーム
- 多数のテキストボックスを手作業で配置したくないユーザー

## 主な機能

- 単一ファイル `.ppt.html` の再生と共有
- タイトル、本文、画像、比較、カード、データ、表、コードの視覚編集
- AI JSON インポート
- 12 種類の基本レイアウトと 4 種類のテーマ
- GitHub Actions による Linux、macOS arm64、macOS x64、Windows ビルド

## クイックスタート

```bash
npm install
npm run serve
```

`http://localhost:5173` を開きます。
