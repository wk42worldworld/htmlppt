# PPT.html Studio

PPT.html Studio는 AI 시대를 위한 프레젠테이션 도구입니다. AI가 불안정한 PPTX를 직접 만들 필요가 없고, 사람이 HTML을 직접 작성할 필요도 없습니다. `.ppt.html` 파일 안에 구조화된 JSON을 저장하고, 브라우저에서 슬라이드로 재생하며, 앱에서 다시 편집할 수 있습니다.

## 누구를 위한 도구인가

- AI로 보고서, 강의 자료, 피치덱, 제품 소개를 만드는 사람
- 오프라인으로 열고 공유하며 다시 편집할 수 있는 파일이 필요한 팀
- 수많은 텍스트 박스를 손으로 맞추고 싶지 않은 사용자

## 핵심 기능

- 단일 `.ppt.html` 파일 재생과 공유
- 데스크톱 앱에서 `.ppt.html` 열기, 저장, 다른 이름으로 저장
- 제품 피치, 강의 자료, 프로젝트 보고용 시작 템플릿
- 제목, 본문, 이미지, 비교, 카드, 데이터, 표, 코드 시각 편집
- AI JSON 가져오기
- 사람과 AI가 모두 읽을 수 있는 검사 보고서
- 12개 기본 레이아웃과 4개 테마
- GitHub Actions로 Linux, macOS arm64, macOS x64, Windows 빌드

## 빠른 시작

```bash
npm install
npm run serve
```

`http://localhost:5173` 을 엽니다.
