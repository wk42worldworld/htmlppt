# PPT.html Studio

PPT.html Studio는 AI 시대를 위한 프레젠테이션 도구입니다. AI가 불안정한 PPTX를 직접 만들 필요가 없고, 사람이 HTML을 직접 작성할 필요도 없습니다. `.ppt.html` 파일 안에 구조화된 JSON을 저장하고, 브라우저에서 슬라이드로 재생하며, 앱에서 다시 편집할 수 있습니다.

## 새 기능

v0.2.15는 일반 PPT 사용자가 자주 만지는 발표와 편집 기본기를 더 다듬었습니다. 삽입 팔레트에 검색 필터를 추가해 차트, 표, KPI, 미디어, 타임라인 같은 컴포넌트를 빠르게 찾을 수 있습니다. 발표 모드에서 `맞춤` / `채우기` 를 전환할 수 있고, 내보낸 단일 파일 플레이어도 `M` 단축키로 같은 전환을 지원합니다. 플레이어와 데스크톱 Help 메뉴에 단축키 안내를 추가했고, 세로형 터치 화면에서 컨트롤을 보이게 할 때 실수로 다음 장으로 넘어가지 않게 했습니다. 발표 중 `Shift+F5` 는 현재 표시 중인 슬라이드를 기준으로 하며, 오른쪽 개체 패널의 삭제 버튼도 키보드 Delete 와 같은 다중 선택 삭제 경로를 사용합니다. Release workflow 는 Node.js 24 로 업데이트했습니다.

v0.2.14는 AI와 사람의 공동 편집 흐름을 강화했습니다. 캔버스 개체는 중복 id, 위치와 크기, 미디어, 차트, 표, 카드, 지표, 타임라인, 인용, 코드, 비교 데이터를 검증합니다. 이미지, 비디오, 오디오, 차트, 표, 카드, 지표, 타임라인, 비교, 인용, 코드, 도형 개체를 선택하면 Typed Inspector 로 주요 필드를 편집할 수 있고, 품질 검사에서 완전한 AI 수정 Prompt 를 복사할 수 있습니다. `htmlppt` Codex skill 과 deck CLI 도 agent 연동의 공식 경로로 정리했습니다.

v0.2.10은 처음 실행했을 때의 완성도를 다듬었습니다. HTML 슬라이드 앱임을 더 쉽게 알아볼 수 있는 새 아이콘, 편집기와 내보낸 단일 `.ppt.html` 파일의 실제 favicon, 더 넓은 기본 데스크톱 창, 1280px 폭에서 덜 답답한 열 구성, 정리된 현재 슬라이드 작업 막대, 그리고 실행 중 기본 Electron 브랜딩이 보이는 시간을 줄이기 위한 더 이른 앱 식별 설정을 포함합니다.

v0.2.9에는 첫 번째 요소 스타일 패널이 추가되었습니다. 캔버스에서 제목, 본문, 텍스트 상자, 카드, 표 셀, 지표, 차트 범례, 미디어 블록을 클릭한 뒤 오른쪽 패널에서 글자 크기, 글자색, 정렬, 굵게, 기울임, 배경, 테두리, 모서리, 불투명도를 조정할 수 있습니다. 결과는 구조화된 `styles[path]` 데이터로 저장되고, 내보낸 `.ppt.html` 파일에도 유지되며, AI도 계속 읽고 수정할 수 있습니다.

v0.2.8은 텍스트 삽입과 편집 반응성을 개선했습니다. `텍스트` 를 클릭하면 텍스트 레이아웃으로 조용히 바뀌는 대신 현재 슬라이드에 보이는 캔버스 텍스트 상자가 추가됩니다. 텍스트 상자는 기본적으로 제목을 가리지 않는 위치에 놓이고, 즉시 편집, 드래그, 크기 조절, 방향키 미세 조정, Delete/Backspace 삭제가 가능하며 구조화된 `textBoxes` 데이터로 `.ppt.html`에 저장됩니다. 오른쪽 패널 입력도 프레임 단위로 묶어 렌더링하고 초안 저장을 지연해 더 부드럽게 동작합니다.

v0.2.7은 `.ppt.html`을 진짜 단일 파일 컨테이너로 다룹니다. 저장 또는 다운로드 전에 외부 이미지, 비디오, 비디오 포스터, 오디오를 Data URI로 같은 HTML 파일 안에 포함합니다. 오디오도 삽입, 드래그, 더블클릭 교체가 가능한 정식 컴포넌트가 되었습니다.

v0.2.6에는 메인 편집기에 눈에 보이는 컴포넌트 팔레트가 추가되었습니다. 텍스트, 이미지, 비디오, 차트, 표, 카드, 지표, 타임라인, 인용, 코드 컴포넌트를 클릭하거나 드래그해 현재 슬라이드에 넣을 수 있습니다. 주요 도구 모음 명령은 아이콘 중심으로 바뀌었고 hover 설명을 제공합니다. 비디오는 정식 슬라이드 콘텐츠로 지원되며 앱 UI는 중국어, 영어, 일본어, 한국어로 전환할 수 있습니다.

v0.2.5는 OpenHuman 같은 로컬 우선 데스크톱 AI 앱의 깔끔한 UI 감각을 참고해 편집기 전체 시각 디자인을 새로 다듬었습니다. 조밀한 어두운 상단 바, 더 가벼운 패널, 정돈된 썸네일, 폼, 버튼, 대화상자, 차분한 캔버스 그리드, 부드러운 그림자, 좁은 화면 대응 개선을 포함합니다.

v0.2.4에는 캔버스 편집에 선택과 변형 컨트롤이 추가되었습니다. 편집 가능한 요소를 클릭하면 선택 프레임이 표시되고, 드래그 이동, 8개 핸들 크기 조절, 방향키 미세 조정, `Shift` 빠른 미세 조정, 현재 요소 위치와 크기 초기화를 할 수 있습니다. 결과는 구조화된 `canvas` `x/y/w/h` 데이터로 저장되며 내보낸 `.ppt.html` 파일에서도 유지됩니다.

v0.2.3에는 첫 번째 캔버스형 편집 레이어가 추가되었습니다. 캔버스에서 제목, 본문, 카드, 지표, 표 셀, 차트 범례, 코드 블록을 더블클릭해 직접 수정할 수 있습니다. 편집 가능한 요소는 가볍게 드래그해 위치를 미세 조정할 수 있고, 위치는 구조화된 `canvas` 오프셋으로 저장됩니다. 이미지를 캔버스로 드롭하거나 썸네일을 드래그해 슬라이드 순서를 바꿀 수도 있습니다.

v0.2.2에서는 전체 화면 발표 스케일링을 개선해 16:9 화면에서 예전의 검은 여백 없이 슬라이드가 화면을 채웁니다. v0.2.1에는 구조화된 차트 슬라이드가 추가되었습니다. 막대, 선, 도넛 차트를 시각적으로 편집할 수 있고, AI도 같은 `chart` JSON을 생성할 수 있습니다.

제품의 기본 원칙은 간단합니다. AI는 구조를 쓰고, 사람은 판단하며, 렌더러는 안정적인 레이아웃을 유지합니다.

## 누구를 위한 도구인가

- AI로 보고서, 강의 자료, 피치덱, 제품 소개를 만드는 사람
- 오프라인으로 열고 공유하며 다시 편집할 수 있는 파일이 필요한 팀
- 수많은 텍스트 박스를 손으로 맞추고 싶지 않은 사용자

## 핵심 기능

- 단일 `.ppt.html` 파일 재생과 공유. 이미지, 비디오, 오디오, 포스터, CSS, 플레이어 JavaScript가 한 파일에 포함됨
- 데스크톱 앱에서 `.ppt.html` 열기, 저장, 다른 이름으로 저장
- 제품 피치, 강의 자료, 프로젝트 보고용 시작 템플릿
- 제목, 본문, 이미지, 비교, 카드, 데이터, 차트, 표, 코드 시각 편집
- 메인 화면 컴포넌트 팔레트에서 텍스트, 이미지, 비디오, 오디오와 여러 비교, 차트, 표, 카드, 지표, 타임라인 변형을 클릭하거나 드래그해 삽입
- 캔버스 더블클릭 편집, 요소 드래그 이동과 크기 조절, 키보드 미세 조정, 이미지 드롭, 썸네일 드래그 순서 변경
- 이미지는 로컬 포함, cover/contain 맞춤, 캡션, 전체 이미지 슬라이드, 배경 이미지 슬라이드를 지원
- 비디오는 로컬 포함, URL / Data URI, 포스터, 캡션, cover/contain 맞춤을 지원
- 오디오는 로컬 포함, URL / Data URI, 캡션, 브라우저 기본 컨트롤을 지원
- 차트는 구조화된 라벨과 시리즈로 막대, 선, 도넛 형태를 렌더링
- 발표 모드와 내보낸 단일 파일 플레이어는 없음, 페이드, 슬라이드, 밀기, 줌 전환 효과를 지원
- 앱 UI는 중국어, 영어, 일본어, 한국어 전환을 지원
- AI JSON 가져오기
- `.ppt.html` 검증, 추출, 빌드, 수정을 위한 Agent CLI 와 Codex skill
- 사람과 AI가 모두 읽을 수 있는 검사 보고서
- 17개 기본 레이아웃과 4개 테마
- GitHub Actions로 Linux, macOS arm64, macOS x64, Windows 빌드, 선택적 macOS 서명 및 공증 지원

## 빠른 시작

```bash
npm install
npm run serve
```

`http://localhost:5173` 을 엽니다.

## 5분 튜토리얼

1. PPT.html Studio를 엽니다.
2. `模板` 을 클릭하고 제품 피치, 강의 자료, 프로젝트 보고, 또는 demo deck을 선택합니다.
3. 오른쪽 패널에서 제목, 부제목, 지표, 차트, 표, 이미지, 발표 메모를 수정합니다. 캔버스에서는 직접 더블클릭, 드래그, 크기 조절, 키보드 미세 조정도 할 수 있습니다.
4. AI 초안을 사용할 때는 `AI JSON` 을 클릭합니다. 일반 JSON과 `json` 코드 블록을 모두 읽을 수 있습니다.
5. `检查` 를 클릭합니다. 오류나 경고가 있으면 검사 보고서를 AI에게 전달해 수정할 수 있습니다.
6. `演示` 로 발표 화면을 미리 봅니다.
7. 브라우저 버전에서는 `保存 / 下载`, 데스크톱 앱에서는 `保存` / `另存为` 를 사용해 공유 가능한 `.ppt.html` 파일을 만듭니다.
